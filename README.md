

### **Desplegable de cargar JSON**
Crear una página HTML que contenga dos desplegables, uno para las provincias y otro para los municipios.  
Al comenzar a ejecutarse la página deberán cargarse desde el archivo **cargaProvinciasJSON.php** las provincias en el desplegable correspondiente.  
Después, al seleccionar una provincia deberán cargarse en el otro desplegable los municipios correspondientes a esa provincia, que deberemos obtener desde el archivo **cargaMunicipiosJSON.php**.



> **Crear una página web que consuma la API de OpenTDB (https://opentdb.com/api.php?amount=1&type=multiple) para mostrar preguntas tipo test.**  
> - La página debe mostrar una pregunta y cuatro posibles respuestas (una correcta y tres incorrectas).  
> - El usuario selecciona una respuesta:  
>   - Si acierta, pasa automáticamente a la siguiente pregunta.  
>   - Si falla, se muestra cuántas preguntas consecutivas había contestado correctamente hasta ese momento.  
> - La API permite configurar cantidad de preguntas, categoría y dificultad (mediante https://opentdb.com/api_config.php).  
> - Se puede hacer una variante del juego con más de una pregunta y traducir los textos de la interfaz al español.
