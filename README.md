
### **6**
Crear una página HTML que contenga dos desplegables, uno para las provincias y otro para los municipios.  
Al comenzar a ejecutarse la página deberán cargarse desde el archivo **cargaProvinciasXML.php** las provincias en el desplegable correspondiente.  
Después, al seleccionar una provincia deberán cargarse en el otro desplegable los municipios correspondientes a esa provincia, que deberemos obtener desde el archivo **cargaMunicipiosXML.php**.
